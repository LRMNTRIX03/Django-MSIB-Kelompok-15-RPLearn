Tentang SkillPath

Lewat SkillPath kalian bisa mengambil Sertifikasi.
Di SkillPath Front-End Web Developer ini, kalian akan diberikan sebuah tugas untuk membuat website sederhana menggunakan HTML, CSS, dan JavaScript dasar.
Syarat untuk memulai SkillPath ini adalah kalian harus terlebih dahulu mendaftar dan menyelesaikan kelas HTML, CSS, dan JavaScript dasar.
Setelah menyelesaikan SkillPath ini, kalian akan mendapatkan SkilBadge sesuai dengan nilai dari hasil tugas akhir kalian.


Persiapan SkillPath

  • Gunakan komputer atau laptop dengan koneksi internet yang stabil.
  • Install browser (disarankan menggunakan Chrome).
  • Mendaftar dan menyelesaikan kelas HTML, CSS, dan JavaScript Dasar.
